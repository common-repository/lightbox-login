== Description ==

Lightbox Login is the easy way to have your users sign in to any page or post. Simply embed a shortcode and your readers will be prompted to login before reading the page.

== Installation ==

1. Upload `lightbox-login.zip` to the `/wp-content/plugins/` directory or search for plugins and find us in the WordPress.org directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the shortcode `[lightbox_login]` to any page or post!

== Frequently asked questions ==

= Should I use this plugin with sensitive information =

No. This plugin prompts the user to login but does not force a login.
